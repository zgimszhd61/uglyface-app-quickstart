# uglyface-app-quickstart
```
请帮我生成一个采用了一种简约而夸张的风格的头像，主要特点如下：
1. 线条简洁：头像的线条非常简洁，头发、脸部轮廓和五官都是用简单的线条勾勒出来的，没有复杂的细节。
2. 夸张的表情：头像的表情非常夸张，眼睛大且圆，嘴巴呈现出惊讶或困惑的表情，增强了表情的戏剧性。
3. 颜色使用单一：整个头像的颜色非常少，主要是肤色、黑色和淡黄色，背景是单一的灰色，使得头像的重点更加突出。
4. 头发凌乱：头发线条随意且凌乱，给人一种不修边幅的感觉。
5. 类似于涂鸦或草图，强调情感表达和视觉冲击力，适合用于表现幽默或轻松的主题。
6. 脸部朝向为正面
```
